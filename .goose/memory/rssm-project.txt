# # rssm project common-lisp overviewrssm tooling roswell ocicl testing

RSSM (RSS Manager) project at /home/skin/Code/djha-skin/rssm. Common Lisp CLI tool for converting RSS feed lists between Newsboat urls format, RSSSavvy JSON, and OPML. Stack: Roswell, OCICL (NOT Quicklisp), ASDF, CLIFF (CLI), Parachute (tests), Serapeum, Plump (OPML XML). Package: com.djhaskin.rssm. Reverse-domain naming convention. 80-char line limit on ALL files (.lisp, .md, .ros). One-level folders only — no nesting.RSSM tooling rules: Always use `ros +Q` to disable Quicklisp (never omit +Q). One-shot Lisp evaluation: tools/one-shot-lisp.ros '(expr)'. REPL in tmux pane 0 (ros run), Goose in pane 1, manual in pane 2. Tests use Parachute: define-test, is, true, false, fail, of-type, finish. Run tests from shell: ros +Q -- --eval '(asdf:test-system "com.djhaskin.rssm")' --eval '(uiop:quit)'. Add dependencies with: ocicl install <name>. If weird compile errors: rm -rf ocicl/ && ocicl install.
